## **Session 1  —  Project Introduction and Development Environment Setup** 

**Objective** 

Understand the project and prepare your local development environment so you can start building the application. 

## **1.  What Is This Project?** 

You will build a web application called the AI-Powered Exam Preparation Planner. It allows a student to manage their academic modules and then use artificial intelligence to generate a personalized 1-week revision plan for exams. 

The application is a CRUD project with AI integration: 

- Create — add a new module to the database 

- Read — display all modules 

- Update — edit a module 

- Delete — remove a module 

- AI — generate a smart revision plan based on module data 

For each module, the student enters: 

- Module name and teacher 

- Difficulty (Easy / Medium / Hard) 

- Career importance (Low / Medium / High) 

- Current progress (0–100 %) 

- Understanding level (Low / Medium / High) 

- Exam date

## **3.  What Is Localhost?** 

Localhost means running a website on your own computer. You do not need to upload anything to the internet. WampServer or XAMPP creates a local web server (Apache) on your machine so you can test your PHP pages at: 

## **http://localhost/exam_planner** 

## **4.  Project Folder Structure** 

Place your project inside the WampServer / XAMPP web root: 

htdocs/  (XAMPP)  or  www/  (WAMP) exam_planner/ index.php css/  style.css includes/  db.php  header.php  footer.php 

## **5.  Practical Tasks This Week** 

- **1** Install WampServer or XAMPP on your computer **2** Start Apache and MySQL from the control panel 

- **3** Create your project folder inside htdocs or www 

- **4** Create a simple index.php that displays 'Hello World' 

- **5** Open it in your browser at http://localhost/exam_planner **6** Create a basic homepage with HTML5 and add simple CSS styling 

## **Expected Result by End of Week** 

> ✓ WampServer or XAMPP is installed and running 

> ✓ You can access your project at http://localhost/exam_planner 

> ✓ A basic homepage is displayed correctly in the browser 

## **Advice & Common Mistakes to Avoid** 

**!** Make sure both Apache and MySQL are green (running) before opening the browser 

**!** Keep your project folder name short and without spaces 

**!** Do not start the AI integration yet — get the base setup working first 

**!** If localhost does not load, check that port 80 is not blocked by another application
